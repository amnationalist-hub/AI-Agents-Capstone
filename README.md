# ResearchGPT: Enterprise Research Automation Agent

**Google 5-Day AI Agents Intensive - Capstone Project**

[![Kaggle](https://img.shields.io/badge/Kaggle-Notebook-blue)](https://www.kaggle.com/code/nationalist313/ai-agent-capstone-project-1)



## Overview

ResearchGPT is an enterprise-grade AI agent system that automates comprehensive research workflows using Google's Agent Development Kit (ADK).

### Key Features
- Multi-agent orchestration with 4 specialized agents
- Generates reports in 30 seconds (vs. 5-10 hours manually)
- Automated quality validation
- Production-ready with error handling and metrics
- Enterprise-focused use cases

## Quick Start

### Prerequisites
- Google Gemini API key
- Python 3.10+
- Kaggle account (for notebook)

### Run on Kaggle
1. Open the notebook: https://www.kaggle.com/code/nationalist313/capstone-project-1
2. Add your API key in Kaggle Secrets
3. Run all cells (from top to bottom, sequentially)
4. Generate research reports!

## Architecture
```
Planning Agent → Research Agent → Quality Agent → Synthesis Agent
       ↓              ↓               ↓                ↓
   Strategy    Information      Validation         Report
```
